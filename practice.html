Example of inline elements in html

1. HTML inline level elements can appear in the body of an HTML page.

2. It contain data and otherinline elements.

3. By default, inline elements do not begin on new lines.

4. inline elements create shorter structures(than block level elements)

List of inline elements

  • b, i, small
  • em, strong
  • a, br, img, span, sub, sup
  • button, input, label

Example of Block Elements

List of block elements

  • header, footer, ol, ul, li
  • figure, figcaption, form, div, p, table
  • article, aside, h1-h6, main, nav, section

CSS inline, inline-block & block

inline The element doesn’t start on a new line and only occupy just the width it requires. You can’t set the width or height.

inline-block It’s formatted just like the inline element, where it doesn’t start on a new line. BUT, you can set width and height values.

block The element will start on a new line and occupy the full width available. And you can set width and height values.
